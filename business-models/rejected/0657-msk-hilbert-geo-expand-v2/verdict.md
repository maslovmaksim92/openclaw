# 06-verdict

[GEO-EXPAND] 0657 MSK Hilbert GEO Expand v2 — REJECTED: 0/100 | локальный спрос размазан между CRM/CDP/messaging suites, Profit Gate FAIL, отдельный fundable wedge для РФ не доказан.

## Вердикт
**REJECTED**

## Причина
Кейс отклонён на стадии P4 по правилу раннего отклонения.

- Demand = **LOW**
- Profit Gate = **FAIL**
- В РФ уже есть локальные substitutes в CRM/CDP/Telegram automation
- Для Hilbert нет достаточно сильного wedge, который оправдывает отдельный enterprise stack поверх уже существующих систем

## Инвестиционный вывод
Это не fundable geo-expand кейс для РФ в текущем виде. Проблема не в отсутствии самой задачи, а в плохом entry point: боль решается через уже купленные CRM/CDP/messaging suites, а новый иностранный vendor получает длинный sales cycle, compliance friction и слабый шанс быстро выйти на EBITDA >= 500k ₽/мес.

## Что могло бы изменить решение
1. Подтверждённые LOI / design partners из РФ enterprise-сегмента.
2. Явный wedge, который локальные игроки не закрывают: например, уникальный uplift по conversion/revenue с доказуемым ROI.
3. Local-first delivery: on-prem / sovereign cloud / русскоязычная команда внедрения.
4. Публичные кейсы в РФ или CIS с measurable payback < 12 месяцев.
