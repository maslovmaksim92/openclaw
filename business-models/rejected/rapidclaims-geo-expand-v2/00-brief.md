# RapidClaims, geo-expand сигнал по autonomous medical coding

## Суть сигнала
RapidClaims показывает, что autonomous medical coding и revenue cycle automation можно рассматривать как самостоятельный кандидат на новый цикл полной аналитики. Несмотря на прежнее решение открыть другой кейс, текущий resurrection-файл требует отдельного кейса для переоценки с нуля.

## Почему берём в работу
- Есть признаки high-LTV enterprise-модели в медконтуре.
- Ниша сложная для локализации, а значит может поддерживать дорогой сервисный слой.
- В исходном triage уже зафиксированы сильные продуктовые и операционные метрики, достаточные для прохождения intake.

## Следующий шаг
Передать кейс в P3-demand-validation и далее по полному пайплайну P3→P7.
