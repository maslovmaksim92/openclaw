## КАК ЧИТАТЬ ЭТОТ ФАЙЛ

Каждый покупатель описан по схеме:
1. **Реквизиты** (ИНН, ОГРН, адрес, директор, контакты)
2. **Финансовые масштабы** (выручка, прибыль, бюджет на стройматериалы)
3. **Что закупают** (конкретные позиции с ОКПД2)
4. **Сколько закупают** (объёмы в тоннах/штуках/деньгах)
5. **Как закупают** (ЕИС, Росэлторг, прямые договоры, рамочные)
6. **Точка входа** (как начать с ними работать)

Покупатели сгруппированы по приоритету: A+ (самые крупные), A (важные), B (поддержка).
