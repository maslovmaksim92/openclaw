# Источники заказов на строительные подряды в ДНР

Дата: 2026-04-22/23 (MSK)
Статус: рабочая research-карта для маркетинга и системного сбора лидов

## Цель

Собрать максимально широкий и практически полезный список источников, где могут появляться:
- строительные подряды,
- генподряд,
- субподряд,
- капремонт,
- реконструкция,
- благоустройство,
- инженерные сети,
- снабжение стройматериалами,
- поставка+монтаж,
- тендеры и квазитендеры,
- коммерческие закупки,
- прямые сигналы от заказчиков,
- локальные объявления и Telegram-каналы,

по гео **ДНР**.

Главный принцип: для маркетинга нужен не один сайт, а многослойная система мониторинга.

---

## Как читать этот список

### Оценка качества
- **A** — обязательно мониторить, высокий шанс реальных лидов
- **A-** — очень полезный источник, почти ядро
- **B+** — сильный дополнительный источник
- **B** — рабочий дополнительный слой
- **B-** — можно использовать точечно
- **C+ / C** — длинный хвост, радар, косвенные сигналы

### Типы источников
- **Госзакупки**
- **Агрегатор**
- **ЭТП**
- **Подрядная площадка**
- **Telegram / соцсети**
- **Каталог / карта**
- **Прямой заказчик**
- **Новостной/инфо-радар**
- **Поисковый вход**

## 6. Что ещё нужно собирать поверх этих сайтов

Один список сайтов не решает задачу полностью. Для сильного маркетинга нужно собирать ещё 5 сущностей:

1. **Список заказчиков**
   - министерства
   - администрации
   - коммунальные предприятия
   - застройщики
   - девелоперы
   - крупные подрядчики и генподрядчики

2. **Список ключевых слов** по видам работ
   - общестрой
   - инженерка
   - благоустройство
   - дорожные работы
   - фасады/кровля
   - отделка
   - поставка+монтаж
   - техобследование

3. **Список Telegram-каналов и групп**
   - по стройке
   - по объявлениям
   - по Мариуполю
   - по восстановлению
   - по недвижимости/ремонту

4. **Список прямых компаний для outbound**
   - подрядчики
   - застройщики
   - УК
   - поставщики
   - производственные базы

5. **История по выигранным закупкам**
   - кто чаще выигрывает
   - кто регулярно публикует
   - где лучше заходить как субподрядчик, а где как поставщик

### 10.6. Сценарий маркетинга на основе этой карты

1. Сначала ловим **публикацию закупки**.
2. Потом фиксируем **заказчика и тип объекта**.
3. После итогов ищем **победителя**.
4. Затем делаем отдельный каскад outreach:
   - заказчику как потенциальный исполнитель,
   - победителю как субподрядчик,
   - поставщикам и смежникам как партнёрам,
   - локальным Telegram-каналам как радару по фактическому старту работ.
5. Через 2-6 недель возвращаемся к объекту и ищем вторую волну: окна, фасады, кровля, отделка, благоустройство, инженерка.
